Search¶

To enable search in comics you need to install django-haystack and setup one of their search backends. Consult the Haystack documentation for information on further settings.

Furthermore INSTALLED_APPS needs to contain haystack and comics.search. E.g. in your comics/settings/local.py, add:

from comics.settings.base import INSTALLED_APPS
INSTALLED_APPS += ('haystack', 'comics.search')

Solr schema¶

Solr needs to be setup with a schema.xml that knows about the comics comics.core.models.Image model. This file can be created by running:

python manage.py build_solr_schema

Indexing¶

For initial indexing, run:

python manage.py update_index

Subsequent indexing should be done from cron with for instance:

python manage.py update_index --age=HOURS

where HOURS should be replaced according with how often you reindex.
