comics¶

comics is a web based aggregator for web comics. It was originally developed as a tailored web application for http://comics.jodal.no/, but the source code has later been generalized and released for others to change and use. comics is maintained by Stein Magnus Jodal, with comments and input from numerous users. The software is freely available under the GPLv2 license.
